Win10-WSL安装Ubuntu配置开发环境

公司不能使用 macos 进行开发,无奈之下准备使用 win10 安装 wsl2/ubuntu 进行开发环境的搭建

一 基本安装

先确定 BIOS 是否开启虚拟化支持

安装 Windows Terminal和 Ubuntu20.04

  • 找到 windows terminal 地址
  • 创建快捷方式复制到开始菜单
  • 对快捷方式点击右键-属性-快捷方式-高级-用管理员身份运行 .这样以后每次默认使用管理员打开 Windows Terminal,更方便
  • 管理员打开Powershell,开启linux子系统支持
  • 微软商店安装 Ubuntu,进行基本配置

  • 之后可以直接在 Windows Terminal 中打开 Ubuntu,更方便

三 go-wsl2-host

ubuntu 的 ip 会随时变化,使用该程序可动态维护 ubuntu 到 hosts 文件中

  • 下载go-wsl2-host到本地
  • 使用管理员/powershell 打开

四 Docker

  • 先更新系统,换镜像
  • 安装 docker

美化

默认界面太丑,美化也不用太多,配置 ohmyzsh 即可

  • 安装Zsh并将Zsh设置为默认 shell：
  • 安装 ohmyzsh
  • 主题 Powerlevel10k
  • 插件
  • terminal 安装字体
